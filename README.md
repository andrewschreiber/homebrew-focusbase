# Homebrew Tap for Focusbase

This is the official Homebrew tap for [Focusbase](https://focusbase.com), the universal progress notifier.

## Installation

```bash
brew tap andrewschreiber/focusbase
brew install --cask focusbase
```

## Requirements

- **macOS 10.15** (Catalina) or later
- **Apple Silicon** (ARM64) - Intel Macs are not currently supported

## Updating

```bash
brew upgrade --cask focusbase
```

## Uninstalling

```bash
brew uninstall --cask focusbase
```

To also remove user data and preferences:

```bash
brew uninstall --cask --zap focusbase
```
